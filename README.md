## Project Overview
This repository contains my complete Digital Literacy Portfolio, developed as part of the CSE0001 course. Operating under the scenario of a Student Digital Ambassador, this project comprises five distinct modules designed to build and demonstrate proficiency in digital communication, professional online presence, cloud collaboration, and cybercrime prevention.

## Repository Structure & Module Summaries

* 📁 **`report/`**
  * Contains the final compiled Project Report, documenting reflections, observations, and tool analyses for all five tasks.
  
* 📁 **`task-1-presentation/`**
  * Includes a digital literacy awareness infographic designed to educate peers on essential digital tools, safe internet practices, and the meaning of digital citizenship.

* 📁 **`task-2-portfolio/`**
  * Contains visual proof of foundational professional profiles established across key platforms, including GitHub, LinkedIn, and Kaggle, to build a strong digital footprint.

* 📁 **`task-3-platforms/`**
  * Showcases practical familiarity with coding assessment platforms (HackerRank) and cloud collaboration tools (Google Workspace).
  * 🔗 **https://docs.google.com/forms/d/e/1FAIpQLSf6g6-bws9fkHYvDizb_m8LXwNka6DmZqoI1A7KR0aUIKE2RQ/viewform?usp=dialog**

* 📁 **`task-4-email-etiquette/`**
  * Features drafts of professional communications, including an academic extension request and a corporate internship inquiry, alongside a practical social media Do's and Don'ts checklist.

* 📁 **`task-5-cybercrime/`**
  * Includes a detailed case study analyzing the mechanics of UPI and screen-sharing fraud, accompanied by an actionable "Stay Safe Online" prevention checklist tailored for university students.
